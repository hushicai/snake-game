# 命令行贪吃蛇游戏

一个在终端中运行的经典贪吃蛇游戏，使用 Node.js 和原生终端控制实现。

## 特性
- 20×20 游戏板，ASCII 字符显示
- 蛇使用 `●` (头部) 和 `█` (身体) 表示
- 食物使用 `★` 表示
- WASD 或方向键控制移动
- 实时得分显示，每吃一个食物得 10 分
- 游戏速度随得分增加而加快
- 游戏结束画面和重新开始功能

## 安装和运行

### 前提条件
- Node.js v14 或更高版本
- 支持 ANSI 转义码的终端

### 运行游戏
```bash
# 克隆项目后
cd snake-game
npm start
```

或者直接运行：
```bash
node src/index.js
```

### 测试
```bash
npm test
```

## 游戏控制
- **W/↑**: 向上移动
- **S/↓**: 向下移动
- **A/←**: 向左移动
- **D/→**: 向右移动
- **Q**: 退出游戏
- **R**: 重新开始游戏
- **Ctrl+C**: 强制退出

## 游戏规则
1. 蛇会自动向前移动
2. 使用方向键控制蛇的移动方向
3. 吃到食物 (`★`) 后得分增加，蛇身变长
4. 撞到墙壁或自己的身体时游戏结束
5. 游戏速度随得分增加而加快

## 技术细节
- 使用 ANSI 转义码控制终端显示
- 使用 Node.js 的 `readline` 模块处理键盘输入
- 模块化设计，易于扩展和维护

## 已知限制
- 依赖终端对 ANSI 转义码的支持
- 游戏速度受终端性能影响
- Windows 终端可能需要额外配置
