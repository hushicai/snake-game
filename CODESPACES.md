# 在 GitHub Codespaces 中运行贪吃蛇游戏

这个项目可以在 GitHub Codespaces 中直接运行，无需本地安装。

## 如何使用

1. 点击仓库页面上的 "Code" 按钮
2. 选择 "Codespaces" 标签
3. 点击 "New codespace" 创建一个新的开发环境
4. 等待 Codespaces 初始化完成（通常需要几分钟）
5. 在终端中运行以下命令启动游戏：

```bash
npm start
```

或者直接运行：

```bash
node src/index.js
```

## 控制说明

- **W/↑**: 向上移动
- **S/↓**: 向下移动
- **A/←**: 向左移动
- **D/→**: 向右移动
- **Q**: 退出游戏
- **R**: 重新开始游戏
- **Ctrl+C**: 强制退出

## 运行测试

```bash
npm test
```

享受游戏吧！
